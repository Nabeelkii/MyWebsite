Embedded Systems

About the project

The project had multiple phases, the first phase is controlling the LED from the website.
This was done using a microcontroller and a wifi module called an ESP32 to wirelessly control the LEDs on a website.
The final phase was controlling the mobile moving platform from the website such that it was able to follow a black line. I was unable to fully complete the final phase, however I did get it working to a certain extent.

The Demo

The video above shows the final phase, which makes use of a camera to detect the black line and follow it. It then stops moving when it does not detect any black lines.
