Limo Project

About the project

This project was done using the agilex Limo robot. Using Ros, the robot is able to navigate from point to point. The project made use of gmapping and vslam to achieve this. This project also involved designing the area where the robot moved.Each team was assigned a small area. I designed and fabricated the building that was part of the area the team was assigned.
